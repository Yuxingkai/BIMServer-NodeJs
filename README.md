# Three.js-WebGl-View
基于Three.js-WebGl-View

# 使用
1.npm install || yarn install

2.npm run dev 

3.超级玛丽 模式操作 w、a、s、d 空格 (只做了向下的碰撞检测)

# SloceJson 使用
1.model 文件夹为原始模型文件

2.output 切割完后的模型文件

3.终端进入 该目录 node sliceJson.js

# 文档目录
1.src 源码

2.sliceJson 把大的模型文件切割成小模型  

3.static 静态资源 

4.config webpack 配置
